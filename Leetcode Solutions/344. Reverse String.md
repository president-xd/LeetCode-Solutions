## Solution

### C++ = 
```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        vector<char> reverse;
        for (int i = s.size() - 1; i >= 0; i--){
            reverse.push_back(s[i]);
        }
        s = reverse;
    }
};
```
```python
class Solution(object):
    def reverseString(self, s):
        reverse = []
        for i in range(len(s) - 1, -1, -1):
            reverse.append(s[i])
        s[:] = reverse
```
